Cursos

É maravilhoso ver a ampla variedade de cursos relacionados ao desenvolvimento front-end que estou fazendo. Essa jornada de aprendizado está me proporcionando uma base sólida para uma carreira promissora. Aqui está a lista dos cursos que tenho o privilégio de estar cursando:


HTML: Estou construindo a base da estrutura web, compreendendo a importância da marcação semântica. CSS: Aprender a estilizar e dar vida a páginas web é fascinante e crucial. Lógica de Programação: Estou desenvolvendo as habilidades fundamentais para resolver problemas com código de forma eficiente. JavaScript: Estou explorando a linguagem que torna as páginas web dinâmicas e interativas. TypeScript: Continuando minha jornada em JavaScript, estou aprendendo uma tipagem estática que traz mais segurança ao desenvolvimento. Git e GitHub: Estou dominando as ferramentas essenciais para controle de versão e colaboração em projetos. Sistema Web Básico (HTML-CSS-JavaScript): Colocando em prática meu conhecimento básico para criar sistemas web funcionais. React.js: Estou entrando no emocionante mundo dos frameworks de JavaScript para criar interfaces de usuário incríveis. Forms (Formulários): Entendendo como coletar dados de forma eficiente dos usuários. Cloud (Computação em Nuvem): Aprendendo sobre a flexibilidade e escalabilidade da computação em nuvem. Next.js: Explorando uma estrutura de desenvolvimento web que facilita a criação de aplicativos rápidos e eficientes. Testes (Possivelmente Testes Unitários ou Testes de Integração): Compreendendo a importância da garantia de qualidade em código. AWS CloudFront (Serviço de Distribuição de Conteúdo da Amazon Web Services): Entrando no mundo da distribuição de conteúdo em larga escala para sites e aplicativos. Essa variedade de conhecimento está me preparando para enfrentar desafios emocionantes no mundo do desenvolvimento front-end. Estou empolgada para aplicar o que aprendi e continuar minha jornada de aprendizado. Meu entusiasmo e desejo de aprender só crescem à medida que me aproximo de uma carreira de sucesso no desenvolvimento Front-end.
